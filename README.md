# blog
个人博客，利用 [`Hexo`](https://hexo.io) 构建，使用 [Icarus](https://ppoffice.github.io/hexo-theme-icarus/) 主题。
## 如何使用
1. Git 克隆到本地
```shell
git clone git@github.com:imaginefish/blog.git
```
2. 进入仓库目录下，使用 npm 安装 node modules
```shell
cd blog
npm install
```
3. 使用 hexo_cli 进行写作
```shell
hexo new <title>
```
4. 使用 hexo_cli 本地启动服务
```shell
hexo server
```
